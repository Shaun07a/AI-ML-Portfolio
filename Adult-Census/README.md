# Adult Census Income Classification

This project implements multiple Machine Learning classification algorithms on the Adult Census Income Dataset to predict whether an individual's annual income exceeds $50,000.

## Objectives

- Understand and explore the dataset
- Perform data cleaning and preprocessing
- Apply feature engineering techniques
- Train multiple classification models
- Compare model performance using evaluation metrics

## Algorithms Used

1. Logistic Regression
2. Decision Tree
3. Random Forest
4. K-Nearest Neighbors (KNN)
5. Support Vector Machine (SVM)

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- VS Code

## Dataset

Adult Census Income Dataset

Target Variable:
- <=50K
- >50K

## Outcome

The models are trained and evaluated to determine the best-performing algorithm for income classification.
