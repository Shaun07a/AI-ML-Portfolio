# About

This repository serves as a centralized collection of my AI and Machine Learning projects developed throughout my academic journey. The projects demonstrate practical implementations of machine learning algorithms, deep learning architectures, reinforcement learning agents, recommendation systems, and retrieval-augmented generation (RAG) applications.

Each project is organized in its own directory with source code, documentation, datasets (where applicable), and implementation details.
